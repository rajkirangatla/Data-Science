Titanic DataSet : Accuracy Prediction¶

Using this dataset to predict the Survival of Tatanic Passengers using Machine Learning - Logistic Regression algorithm.

Steps Involed.¶
Importing the DataSet into Jupiter Notebook using (pandas library).
Clean the Dataset without missing values and Outliers.
Visualize the DataSet to Understand the features and target objects required to predict.
Using library(sklearn) split the data into testing and training set.
Things to do.¶
Before Proceeding let us understand the story behind the dataset.
Dataset consists of passengers where each of the passengers are given an unique id to identify.
Passengers who Survived is marked as 1 and unsurvived is 0.
Pclass (Passenger Class) Class - 1, Class - 2, and Class - 3.
Name of the each passengers are given in the dataset.
Sex female and male passengers in the dataset.
Passenger Age
SibSp Siblings and Spouse 
Ticket, Fare, and Cabin number for each passenger

# null values in the age column are filled by predicting the age of the passengers according to the class

def age_approx(cols):
    Age = cols[0]
    Pclass = cols[1]
    
    if pd.isnull(Age):
        if Pclass == 1:
            return 38
        elif Pclass == 2:
            return 29
        else:
            return 24
    else:
        return Age
